คำสั่งโรงเรียนวังกระแสวิทยาคม

ที่     /๒๕๖๙

เรื่อง แต่งตั้งและมอบหมายผู้รับผิดชอบงานตามโครงสร้างการบริหารงาน

เพื่อให้การปฏิบัติหน้าที่ราชการภายในโรงเรียนวังกระแสวิทยาคม เป็นไปด้วยความเรียบร้อย มีผู้รับผิดชอบงานในหน้าที่อย่างเป็นระบบ ถูกต้องตามระเบียบ ทำให้การดำเนินงานมีประสิทธิภาพและเกิดผลดีต่อราชการ โดยอาศัยอำนาจตามมาตรา ๓๗ แห่งพระราชบัญญัติระเบียบบริหารราชการกระทรวงศึกษาธิการ พ.ศ.๒๕๔๖ และมาตรา ๒๗ แห่งพระราชบัญญัติระเบียบข้าราชการครูและบุคลากรทางการศึกษา พ.ศ.๒๕๔๗ และแก้ไขเพิ่มเติม (ฉบับที่ ๒) พ.ศ.๒๕๕๑ จึงแต่งตั้งผู้รับผิดชอบงานตามโครงสร้างการบริหารงาน ดังรายละเอียดต่อไปนี้

คณะกรรมการบริหารงานโรงเรียนวังกระแสวิทยาคม
นางนฤมล ทัดสาผู้อำนวยการสถานศึกษาประธานคณะกรรมการ
นางสาวชุติยรัชย์ ทาตะรัตน์รองผู้อำนวยการสถานศึกษารองประธานคณะกรรมการ
นางสุภาวดี วรรณวงศ์ครู / หัวหน้ากลุ่มบริหารงานวิชาการกรรมการ
นางสาวรัตนา บุญรอดครู / หัวหน้ากลุ่มบริหารงานแผนงานและงบประมาณกรรมการ
นายวรสิทธิ์ กิจเจริญครู / หัวหน้ากลุ่มบริหารงานทั่วไปกรรมการ
นายอนุพงศ์ ไชยบุตรครู / หัวหน้ากลุ่มบริหารงานบุคคลกรรมการและเลขานุการ

ผู้อำนวยการสถานศึกษา มีหน้าที่บริหารและกำกับ ติดตาม ดูแล งานบริหารงานวิชาการ งานบริหารงานแผนงานและงบประมาณ งานบริหารงานบุคคล และงานบริหารงานทั่วไป

รองผู้อำนวยการสถานศึกษา มีหน้าที่บริหารและกำกับ ติดตาม ดูแล งานบริหารงานวิชาการ งานบริหารงานแผนงานและงบประมาณ งานบริหารงานบุคคล และงานบริหารงานทั่วไป

หัวหน้ากลุ่มบริหารงานวิชาการ มีหน้าที่ดำเนินการนิเทศ กำกับ ติดตาม ดูแล เกี่ยวกับกลุ่มงานวิชาการ สายงานปฏิบัติการสอน สายงานสนับสนุนการสอน และงานอื่น ๆ ตามที่ได้รับมอบหมาย

หัวหน้ากลุ่มบริหารงานแผนงานและงบประมาณ มีหน้าที่ดำเนินการนิเทศ กำกับ ติดตาม ดูแล เกี่ยวกับกลุ่มงานแผนงาน พัสดุและสินทรัพย์ งานการเงิน บัญชี และงานอื่น ๆ ตามที่ได้รับมอบหมาย

หัวหน้ากลุ่มบริหารงานบุคคล มีหน้าที่ดำเนินการนิเทศ กำกับ ติดตาม ดูแล เกี่ยวกับการบริหารงานบุคคลและงานปกครองข้าราชการครูและบุคลากรทางการศึกษา และงานอื่น ๆ ตามที่ได้รับมอบหมาย

หัวหน้ากลุ่มบริหารงานทั่วไป มีหน้าที่ดำเนินการนิเทศ กำกับ ติดตาม ดูแล เกี่ยวกับกลุ่มงานส่งเสริมนวัตกรรมทางการศึกษา กลุ่มงานบริการ กลุ่มอาคารสถานที่ งานกิจการนักเรียน และงานอื่น ๆ ตามที่ได้รับมอบหมาย

นายอนุพงศ์ ไชยบุตรครูเจ้าหน้าที่แผนงาน
นางสาวเนตรชนก จารุโกนครูธุรการกลุ่มแผนงาน
นายวิชชุกร อินทรเทศครูผู้ช่วยกรรมการ
  • จัดทำแผนปฏิบัติการประจำปี แผนพัฒนาสถานศึกษา รวบรวมและสรุปโครงการจากทุกกลุ่มงาน
๒.๒ การบริหารการเงิน
นางสาวรัตนา บุญรอดครูหัวหน้างานการเงิน
  • เบิก-จ่ายงบประมาณตามระเบียบ จัดทำทะเบียนคุม และรายงานสถานะการใช้จ่าย
๒.๓ การบริหารบัญชี
นางสาวเนตรชนก จารุโกนครูเจ้าหน้าที่บัญชี
  • จัดทำบัญชีและรายงานทางการเงินให้ถูกต้องตามมาตรฐานที่กำหนด
๒.๔ การบริหารพัสดุและสินทรัพย์
นายอนุพงศ์ ไชยบุตรครูหัวหน้าเจ้าหน้าที่พัสดุ
นายวิชชุกร อินทรเทศครูผู้ช่วยเจ้าหน้าที่พัสดุ
  • จัดซื้อจัดจ้าง จัดทำทะเบียนพัสดุ ดูแลรักษาและดำเนินการตรวจรับ/จำหน่ายพัสดุตามระเบียบ
๒.๕ คณะกรรมการตรวจสอบเงิน / เก็บรักษาเงิน
นายวรสิทธิ์ กิจเจริญครูกรรมการ
นางสาวอนัญชุดา วราคุณาศัยครูกรรมการ
นางสุภาวดี วรรณวงศ์ครูกรรมการ
  • ตรวจสอบเงินและหลักฐานการรับ-จ่าย เก็บรักษาและนำเงินส่งคลังตามระเบียบ
๓. กลุ่มบริหารงานบุคคล
คณะกรรมการกลุ่มบริหารงานบุคคล
นางนฤมล ทัดสาผู้อำนวยการสถานศึกษาประธานคณะกรรมการ
นางสาวชุติยรัชย์ ทาตะรัตน์รองผู้อำนวยการสถานศึกษารองประธานคณะกรรมการ
นางสาวอนัญชุดา วราคุณาศัยครูกรรมการ
นายรัฐศาสตร์ มลาศาสตร์ครูผู้ช่วยกรรมการ
นายอนุพงศ์ ไชยบุตรครู / หัวหน้ากลุ่มบริหารงานบุคคลกรรมการและเลขานุการ
๓.๑ การวางแผนอัตรากำลัง / การสรรหาและบรรจุแต่งตั้ง
นางสาวอนัญชุดา วราคุณาศัยครูผู้รับผิดชอบ
  • วางแผนอัตรากำลัง ดำเนินการสรรหา บรรจุแต่งตั้ง เปลี่ยนตำแหน่ง และย้ายบุคลากร
๓.๒ การพัฒนาบุคลากร / วิทยฐานะ / ใบอนุญาตประกอบวิชาชีพ / เครื่องราช
นางสาวอนัญชุดา วราคุณาศัยครูผู้รับผิดชอบ
  • ส่งเสริมการพัฒนาบุคลากร ดำเนินการวิทยฐานะ ใบอนุญาตประกอบวิชาชีพ และเครื่องราชอิสริยาภรณ์
๓.๓ การประเมินผลการปฏิบัติงาน (PA) / การเลื่อนเงินเดือน / ขวัญกำลังใจ
นายอนุพงศ์ ไชยบุตรครูหัวหน้างาน
  • ดำเนินการ PA เลื่อนเงินเดือน และส่งเสริมขวัญกำลังใจ ยกย่องเชิดชูเกียรติบุคลากร
๓.๔ วินัยและการรักษาวินัย / การดำเนินการทางวินัย
นายรัฐศาสตร์ มลาศาสตร์ครูผู้ช่วยหัวหน้างาน
  • ดำเนินการเกี่ยวกับวินัย จรรยาบรรณ การลงโทษ การอุทธรณ์ร้องทุกข์ และการออกจากราชการ
๓.๕ งานทะเบียนประวัติ / สารสนเทศบุคคล / การลา / คณะกรรมการสถานศึกษา
นายรัฐศาสตร์ มลาศาสตร์ครูผู้ช่วยหัวหน้างานทะเบียนประวัติ / สารสนเทศบุคคล
นางสาวอนัญชุดา วราคุณาศัยครูงานการลาทุกประเภท
นายอนุพงศ์ ไชยบุตรครู / หัวหน้ากลุ่มบริหารงานบุคคลงานคณะกรรมการสถานศึกษา
  • จัดระบบทะเบียนประวัติ สารสนเทศบุคคล การลาทุกประเภท และงานคณะกรรมการสถานศึกษา
๔. กลุ่มบริหารงานทั่วไป
คณะกรรมการกลุ่มบริหารงานทั่วไป
นางนฤมล ทัดสาผู้อำนวยการสถานศึกษาประธานคณะกรรมการ
นางสาวชุติยรัชย์ ทาตะรัตน์รองผู้อำนวยการสถานศึกษารองประธานคณะกรรมการ
นายชัชพงศ์ ไชยรัตน์ครูกรรมการ
นางสาวสุทธิดา เชื้อชาติครูกรรมการ
นายศิริชัย ลี้พลครูกรรมการ
นางสาวปรารถนา ยุบลพันธุ์ครูผู้ช่วยกรรมการ
นางสาวนภาลัย นาบุญเจ้าหน้าที่ธุรการกรรมการ
นายวรสิทธิ์ กิจเจริญครู / หัวหน้ากลุ่มบริหารงานทั่วไปกรรมการและเลขานุการ
๔.๑ งานอาคารสถานที่ / สภาพแวดล้อม / รักษาความปลอดภัย
นายชัชพงศ์ ไชยรัตน์ครูหัวหน้างาน
นายคมจิต เหลาศรีพนักงานราชการ/นักการผู้ปฏิบัติงาน
นายกิจติ ปาวันนักการภารโรงผู้ปฏิบัติงาน
  • ดูแลบำรุงรักษาอาคารสถานที่ สภาพแวดล้อม ภูมิทัศน์ และระบบรักษาความปลอดภัย
๔.๒ งานนักการภารโรง
นายวรสิทธิ์ กิจเจริญครูหัวหน้างาน
นายคมจิต เหลาศรีพนักงานราชการ/นักการผู้ปฏิบัติงาน
นายกิจติ ปาวันนักการภารโรงผู้ปฏิบัติงาน
  • กำกับดูแลนักการภารโรงในการทำความสะอาด ดูแลรักษาอาคารสถานที่และบริเวณโรงเรียน
  • มอบหมายและตรวจสอบการปฏิบัติงานของนักการภารโรงให้เป็นไปตามที่กำหนด
๔.๓ งานกิจการนักเรียน / งานปกครอง
นายศิริชัย ลี้พลครูหัวหน้างาน
นายชัชพงศ์ ไชยรัตน์ครูกรรมการ
นายวรสิทธิ์ กิจเจริญครูผู้ช่วย
  • บริหารจัดการงานกิจการนักเรียน ระเบียบวินัย แก้ไขพฤติกรรม และงานยานพาหนะ
๔.๔ ระบบดูแลช่วยเหลือนักเรียน
นายศิริชัย ลี้พลครูหัวหน้าระบบ
ครูที่ปรึกษาทุกคนกรรมการร่วม
  • จัดระบบดูแลช่วยเหลือนักเรียนทั้ง ๕ ขั้นตอน และจัดทำสารสนเทศงานกิจการนักเรียน
๔.๕ งาน To Be Number One
นายศิริชัย ลี้พลครูหัวหน้างาน
  • ขับเคลื่อนโครงการ To Be Number One ดูแลชมรม และเตรียมความพร้อมการประกวดทุกระดับ
๔.๖ งานสภานักเรียน / ประชาธิปไตยในสถานศึกษา
นายศิริชัย ลี้พลครูหัวหน้างาน
  • ส่งเสริมกิจกรรมสภานักเรียนและประชาธิปไตยในสถานศึกษา
  • ดำเนินการเลือกตั้งคณะกรรมการสภานักเรียนตามระเบียบที่กำหนด
๔.๗ งานป้องกันยาเสพติด / ป้องกันอุบัติเหตุ / ระบบ SPS
นายวรสิทธิ์ กิจเจริญครูหัวหน้างาน / หัวหน้า SPS
นายชัชพงศ์ ไชยรัตน์ครูกรรมการ
นายศิริชัย ลี้พลครูผู้ช่วย
นางสาวอนัญชุดา วราคุณาศัยครูผู้ช่วย SPS
  • วางแผนและดำเนินกิจกรรมป้องกัน แก้ไขปัญหายาเสพติดและอบายมุขในโรงเรียน
  • บริหารจัดการระบบ SPS (Safety Promotion School) สถานศึกษาปลอดภัย และดำเนินกิจกรรมป้องกันอุบัติเหตุ
  • รายงานผลการดำเนินงานและปรับปรุงอย่างต่อเนื่อง
๔.๘ งานอนามัยโรงเรียน / ส่งเสริมสุขภาพ / อาหารและโภชนาการ
นางสาวสุทธิดา เชื้อชาติครูหัวหน้างาน
  • ดำเนินงานด้านอนามัยโรงเรียน ส่งเสริมสุขภาพนักเรียนและบุคลากรให้เป็นไปตามมาตรฐาน
  • บริหารจัดการงานอาหารกลางวันและโภชนาการของนักเรียน (รวมถึงชาววังคาเฟ่)
  • จัดทำข้อมูลสุขภาพนักเรียนและรายงานผลการดำเนินงาน
๔.๙ งานสหกรณ์โรงเรียน / ธนาคารโรงเรียน
นายวรสิทธิ์ กิจเจริญครูหัวหน้างานสหกรณ์
นางสาวทักษญา วรเชษฐ์ครูหัวหน้างานธนาคารโรงเรียน
  • บริหารจัดการสหกรณ์โรงเรียนและธนาคารโรงเรียนให้เป็นไปตามระเบียบที่กำหนด
  • ส่งเสริมการออมและทักษะทางการเงินให้แก่นักเรียน
๔.๑๐ งานส่งเสริมคุณธรรม จริยธรรม / โรงเรียนสีเขียว / เศรษฐกิจพอเพียง
นายวรสิทธิ์ กิจเจริญครูหัวหน้างาน
นายศิริชัย ลี้พลครูกรรมการ (ส่งเสริมคุณธรรม)
นางมิรัณตี อัมพาครูกรรมการ (เศรษฐกิจพอเพียง)
  • ส่งเสริมและพัฒนาคุณธรรม จริยธรรมของนักเรียนและบุคลากรในสถานศึกษา
  • ขับเคลื่อนกิจกรรมโรงเรียนสีเขียวและสิ่งแวดล้อม
  • บูรณาการปรัชญาเศรษฐกิจพอเพียงในการจัดการเรียนรู้และการดำเนินงานของโรงเรียน
๔.๑๑ งานสารสนเทศโรงเรียน / เทคโนโลยีและประชาสัมพันธ์
นายรัฐศาสตร์ มลาศาสตร์ครูผู้ช่วยหัวหน้างาน
นางสาวอนัญชุดา วราคุณาศัยครูกรรมการ
นางสาวนภาลัย นาบุญเจ้าหน้าที่ธุรการผู้ช่วยประชาสัมพันธ์ / สารสนเทศ
  • บริหารจัดการระบบสารสนเทศของโรงเรียนให้เป็นปัจจุบันและถูกต้อง
  • ดูแลช่องทางประชาสัมพันธ์ (เว็บไซต์ Facebook และสื่อออนไลน์) เผยแพร่ข้อมูลข่าวสารกิจกรรมของโรงเรียน
๔.๑๒ งานสารบรรณ / งานปฏิคม
นางสาวนภาลัย นาบุญเจ้าหน้าที่ธุรการหัวหน้างานสารบรรณ / ปฏิคม
นางสาวธันยกานต์ สีคะพี่เลี้ยงเด็กพิการผู้ช่วยสารบรรณ
  • รับ-ส่ง ลงทะเบียน จัดเก็บ และบริหารงานสารบรรณของโรงเรียนให้ถูกต้องตามระเบียบ
  • ดูแลงานปฏิคมและต้อนรับผู้มาติดต่อราชการ
๔.๑๓ งานโสตทัศนูปกรณ์
นายวิชชุกร อินทรเทศครูผู้ช่วยผู้รับผิดชอบ
  • ดูแล บำรุงรักษา และให้บริการโสตทัศนูปกรณ์ เครื่องเสียง และอุปกรณ์โสตของโรงเรียน
  • สนับสนุนด้านเทคนิคโสตในการจัดกิจกรรมต่าง ๆ ของโรงเรียน
๔.๑๔ การรับนักเรียน
นางสาวทักษญา วรเชษฐ์ครูผู้รับผิดชอบ
  • ดำเนินการรับนักเรียนตามประกาศและแนวนโยบายของสำนักงานเขตพื้นที่การศึกษา
  • จัดทำข้อมูลและรายงานผลการรับนักเรียนต่อผู้บริหารและหน่วยงานที่เกี่ยวข้อง
